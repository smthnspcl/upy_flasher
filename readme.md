#### ... use it
```

{arguments}			{default}

	-b	--binary	firmware/esp32-20180627-v1.9.4-225-gd8dc918d.bin
	-e	--erase		not set / false
	-c	--chip		esp32
	-u	--upload	[]
				-u can be used multiple times for multiple files
	-h	--help
```
